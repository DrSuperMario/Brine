API usage and information

API is located at https://brinenews.herokuapp.com

To get JSON formatted NEWS - https://brinenews.herokuapp.com/newslist
Get data in linux : curl -l http://brinenewsapi.herokuapp.com/newslist
Get data with python: 

            

                import requests

                data = requests.get("http://brinenewsapi.herokuapp.com/newslist")

                print(data.content)
